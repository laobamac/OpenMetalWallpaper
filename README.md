**OpenMetalWallpaper** 是一个开源的 macOS 动态壁纸引擎，使用 Metal + AVFoundation 技术打造，支持多显示器独立播放视频壁纸。完美兼容 Steam Wallpaper Engine 的视频壁纸格式。

> ⚠️ **当前状态**：开发中，仅支持视频格式壁纸文件夹（project.json + 视频文件）。后续将逐步加入 Web、Scene 等更多类型，欢迎 star & pr！

## 功能亮点

- 多显示器独立控制：每块屏幕可设置不同壁纸、不同参数
- 自定义播放体验：音量、倍速、循环、缩放模式（填充/适应/拉伸/自定义偏移+缩放）
- 性能优化选项：支持预加载视频到内存，显著减少循环播放卡顿
- 智能暂停：其他应用全屏或激活时自动暂停，节省 GPU
- Steam 壁纸直接导入：拖入包含 project.json 的文件夹即可使用
- 菜单栏快速操作：暂停/恢复、打开主界面、检查更新
- 持久化配置：每个壁纸 + 每个显示器的设置自动保存
- 自动更新检测：GitHub Releases 最新版本提醒
